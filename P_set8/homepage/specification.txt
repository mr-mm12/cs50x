Specification for Crypto Hub Website

HTML Tags Used (10):
1. <h1> - Main headings
2. <h2> - Subheadings
3. <h3> - Section headings
4. <p> - Paragraphs for text content
5. <a> - Hyperlinks for navigation between pages
6. <img> - Images for exchanges and coins
7. <nav> - Navigation bar
8. <div> - Layout and container sections
9. <ul> - Lists (used in navbar and content)
10. <li> - List items (used in navbar and content)

Bootstrap Features:
- Navbar component with responsive design
- Card component for exchange descriptions
- Buttons with btn, btn-primary, btn-outline-secondary classes
- Grid system using row and col classes for layout
- Progress bars for poll results

JavaScript Usage:
- Poll functionality with interactive selection and submit
- Dynamic updating of progress bars to show results
- Storage of poll results in localStorage
- Handling of form validation and reset buttons

Notes:
- The website contains 5 HTML pages: index.html, introduction.html, buy.html, risks.html, poll.html
- Users can navigate between all pages using hyperlinks
- The site is mobile-friendly and uses responsive Bootstrap layout
- JavaScript provides interactivity for polls and form feedback
